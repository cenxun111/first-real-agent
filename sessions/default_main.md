## 🎯 CNN (卷积神经网络) 核心概念

### 1️⃣ **为什么需要 CNN？**
- **传统神经网络的问题**: 全连接层会将图像展平为一维向量，丢失了空间结构信息
- **CNN 的优势**: 保留图像的二维/三维结构，适合处理图像数据

### 2️⃣ **三大核心组件**

#### 📌 **卷积层 (Convolutional Layer)**
```
输入: 3x3 的滤波器 × 5x5 的特征图 → 输出特征图
作用: 提取局部特征（边缘、纹理、形状等）
特点: 
- 参数共享（同一个滤波器在整个图像上滑动）
- 减少参数量，避免过拟合
```

#### 📌 **池化层 (Pooling Layer)**
```
常见类型: Max Pooling, Average Pooling
作用: 降采样，保留最重要特征
效果: 
- 减小特征图尺寸（如 4x4 → 2x2）
- 增加感受野
- 降低计算量
```

#### 📌 **激活函数 (Activation Function)**
```
常用: ReLU, Leaky ReLU
作用: 引入非线性，让网络能学习复杂模式
公式: f(x) = max(0, x) [ReLU]
```

### 3️⃣ **工作流程**
```
原始图像 
    ↓
卷积层 → 提取特征（边缘、纹理）
    ↓
池化层 → 降采样保留关键信息
    ↓
多次堆叠 → 从低级特征到高级语义特征
    ↓
全连接层 → 分类/回归输出
```

### 4️⃣ **经典 CNN 架构**
- **LeNet-5**: 最早用于手写数字识别
- **AlexNet**: 2012 年 ImageNet 夺冠，开启深度学习时代
- **VGG**: 小卷积核堆叠（3x3）
- **ResNet**: 残差连接解决深层网络退化问题

# 花期预报来了！从早樱初绽到桃李争妍，庐山每周都有新惊喜

[九江新闻网](javascript:;)

当第一缕春风吹过鄱阳湖的水面，庐山的山麓、山腰与山巅，便开始了一场关于色彩的苏醒。2026年的春天，不是悄然来临的——它从方竹寺300年古樱的枝头迸发，从如琴湖玉兰的花苞里绽开，从五老峰下油菜花田的金色波浪里席卷而来。这是一场从2月绵延至4月的盛大花事，而整个3月，免票的庐山，邀你赴约。

**2月下旬–3月上旬：早春的暗香奇卉**

**瑞香｜锦绣谷**

**花期：2月下旬至4月上旬**

锦绣谷的崖壁间，瑞香正悄然绽放。此花异香酷烈，花形如丁香，相传为庐山僧人梦中得佛祖指点寻得，李时珍亦在《本草纲目》中记下这段传奇。香气穿林越谷，未睹花容，先醉其香，是早春最神秘的惊喜。

**梅花｜庐山北门、桃花溪**

**花期：1月下旬–3月中旬**

梅花是春天的信使，此刻已近尾声，却仍是“暗香浮动”的最好注解。沿着桃花溪漫步，疏影横斜，清浅的水面倒映着最后一抹红韵，风过处，香如故。

**3月中旬：早樱与梅，春天的第一行诗**

**早樱｜方竹寺 + 庐山花语世界**

**花期：3月上旬–4月初**

方竹寺内，那棵三百岁的古早樱醒了。粉白的花瓣缀满枝头，与黄色的古墙、黛色的飞檐轻声对话。晨光初透的清晨八点，香火轻绕，花影斑驳，仿佛整个春天的禅意，都凝在了这一树温柔里。

而在庐山花语世界，早樱、海棠与红梅交织成一场色彩的盛宴，粉色、嫣红、雪白——随手一拍，便是春日的封面。

即使偶遇春雨，也不必遗憾——花语世界的温室里，蝴蝶兰、郁金香、君子兰、三角梅、鲁冰花四季常开，为你留住一个不下雨的春天。

**3月中旬–4月上旬：樱花与桃，春色的顶峰时刻**

**樱花｜天花井森林公园 + 蔡家湖畔**

**花期：3月中旬–4月上旬**

天花井的千株樱树，一夜之间汇成粉白色的海。风起时，花瓣如雪纷扬，你在树下，便成了画中人。

蔡家湖畔，临水的樱花正对着碧波梳妆。再过几日，当花影与涟漪重叠，那将是春天最温柔的重逢。

**桃花｜桃花源景区 + 花径公园**

**花期：3月中旬–4月上旬**

沿着溪水走进桃花源，陶渊明笔下的“芳草鲜美，落英缤纷”不再是文字，而是眼前的真实。桃枝探出溪上，花瓣随水流去，每一步都走在诗里。

而花径公园的山桃花，要等到四月才肯盛放——那是白居易的“人间四月芳菲尽，山寺桃花始盛开”，千百年来，依然准时。

**油菜花｜五老峰下、温泉镇**

**花期：2月上旬–4月下旬**

五老峰脚下，油菜花开成了一片金色的海。远山如黛，花田如金，风一吹，浪一波，那是春天最明亮的笑声。

**3月下旬–4月下旬：百花争妍，晚春的压轴盛宴**

**晚樱与海棠｜庐山植物园 + 五老峰**

**花期：3月–4月**

庐山植物园里，晚樱与海棠正悄悄结出花苞。待三月末春风再暖一些，粉白与嫣红将交织成雾，与山间的云雾缠绵，让人分不清是花在云里，还是云在花间。

**玉兰｜如琴湖 + 牯岭街**

**花期：2月下旬–4月上旬**

如琴湖畔，玉兰花开得一尘不染。白色的花瓣映着碧蓝的湖水，远处是老别墅的红顶——那是庐山春日最清雅的定格。

含笑、丁香、二月兰……它们在山径旁、溪水边、古寺后静静开放，等你偶遇，等你俯身，等你为这一缕香气驻足。

**4月中旬–5月中旬：压轴花事，杜鹃与晚樱**

**杜鹃｜五老峰、含鄱口、植物园**

**花期：4月中旬至5月中旬**

杜鹃是庐山春季最值得期待的压轴花事。自4月中旬起，山间杜鹃（含映山红、云锦杜鹃等）次第盛开。无论登五老峰望远，或是在植物园漫步，都能欣赏到云海与花海交织的壮美景色。一簇簇粉紫、嫣红在云雾中若隐若现，宛如仙境。

**关山樱｜小天池观光车站**

**花期：4月中旬至5月上旬**

小天池观光车站的关山樱，重瓣叠粉，如云如霞。微风拂过，花瓣翩然旋落，铺就一地温柔。站在樱花树下，远眺群山，春光在此刻定格成最浪漫的画面。

**实用信息**

免票福利：2026年3月1日–31日，庐山风景区对全球游客免门票。春天免费，只等你来。

交通贴士：南昌已开通“庐山赏花直通车”，一票直达；自驾可沿环山公路行驶，沿途设有临时赏花停车点。

穿搭建议：山中温差大，记得带一件轻便外套和一双好走的鞋。花径很长，春天很短，别让脚程拖慢了心情。

拍照黄金时间：清晨8–10点，傍晚4–6点，光线最温柔，最适合记录花与你的相遇。

文明赏花：花开堪赏直须赏，莫待无花空折枝——但请勿攀折，让美留在枝头，也留给下一个看花的人。

从第一朵樱花的初绽，到最后一片杜鹃的绚烂，庐山用三个月的时间，把春天演绎到了极致。而三月免票的诚意，更让这场花事成为2026年春天最值得奔赴的约定。
